Suspend Auto-Sleep While Playing Media Content Off

Prevents auto-sleep from activating while media content is playing. If this option is disabled, the console will still go to sleep after four hours of media playback.

Maintain Internet Connection in Sleep Mode On

If this option is enabled, downloads initiated from your PC or smartphone will start right away, even when the console is in sleep mode. Disabling this option will reduce the console's energy consumption during sleep mode while connected to the internet.

◆ This will have an effect only when connected to the internet via wired connection.

Wake When AC Adapter Is Disconnected On

If this option is enabled, the console will wake from sleep mode when it is removed from the dock.

If this option is enabled, the Nintendo Switch Pro Controller will communicate with the console via wired communication when connected to it using the USB charging cable. The NFC touchpoint on the Pro Controller will be disabled while the controller is using wired communication.
